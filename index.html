Sobre Mim

Sou desenvolvedor Front-End.

Atualmente sou formado em Engenharia Mecânica pela UNIFRAN(Universidade de Franca) e estou cursando Engenharia da Computação pela FMU(Centro Universitário das Faculdades Metropolitanas Unidas).

Sempre tive interesse pela area de tecnologia, mas nunca havia consiguido me dedicar inteiramente a isso, há 6 meses venho me dedicando e estudando a fundo o Front-End, onde ne um curso paralelo que faço junto a faculdade estou me aprofundando em HTML5, CSS, Java Script e React.
Estou em busca de uma oportunidade para ingressar nesse mundo que ja amo a anos, quero me dedicar inteiramente a ser um desenvolvedor, crescendo e aprendendo cada vez mais.

Minhas Habilidades

Conheça melhor as ferramentas que utilizo em meus projetos.

HTML5

O HTML5 é a versão mais recente da Linguagem de Marcação de Hipertexto, que suporta tags, elementos e conteúdo multimídia.

CSS3

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets (ou simplesmente CSS), pela qual se define estilos para um projeto web (página de internet).

JavaScript

Javascript, ou simplesmente JS, é uma linguagem de programação de uso geral, aplicada principalmente para desenvolvimento web e desenvolvimento de software.

Banco de Dados

Um banco de dados é uma coleção organizada de informações - ou dados - estruturadas, normalmente armazenadas eletronicamente em um sistema de computador.

PHP

O PHP é uma linguagem de programação voltada originalmente para desenvolvimento de sites e aplicações web. Por ser de código aberto e de fácil aprendizado, é uma das mais populares do mundo.
